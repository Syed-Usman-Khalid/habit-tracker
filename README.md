A lightweight Habit Tracker web app built with HTML, CSS, and JavaScript. It allows users to create daily habits, track completion status, and reset progress through a simple and responsive interface designed for clarity and usability. 🗓️ Habit Tracker (Frontend Assessment)

A simple browser-based Habit Tracker built using HTML, CSS, and Vanilla JavaScript. Users can add habits, mark them as completed or not completed, and reset all progress.

🚀 Features Add new habits dynamically Mark habits as completed / not completed Simple visual toggle for tracking progress Reset all habits with one click Fully works in the browser (no backend) Data persists only during session (no database/localStorage if not implemented) 🛠️ Tech Stack HTML5 CSS3 Vanilla JavaScript (ES6)

No frameworks or libraries were used to keep the project lightweight and beginner-friendly. 📦 How to Run This Project Option 1 (Direct Open) Download or clone the repository Open the project folder Double-click on: index.html It will run directly in your browser.
